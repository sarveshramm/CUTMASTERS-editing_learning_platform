1. Basics of photo Editing Software

  • Learn interface and tools of popular editors (e.g., Adobe Premiere Pro, DaVinci Resolve, CapCut, or Filmora).
  • Understand project settings (resolution, frame rate, aspect ratio).
  • Import and organize media files properly.
  • Create timelines and basic sequences.
  • Save/export your first simple video.

2. Framing

  • Use razor tool/split tool to cut clips.
  • the beginning/end of clips precisely.
  • Reorder clips to create story flow.
  • Work with multiple layers (video/audio tracks).
  • Add and manage transitions (fade, dissolve, swipe).

3. Adding Text, Titles

  • Create basic titles and text overlays.
  • Use motion text and templates (Premiere Pro Essential Graphics, etc.).
  • Design lower thirds for intros/interviews.
  • Match fonts and styles to video tone.
  • Animate text (in/out effects).

4. Color Grading

  • Use basic color correction tools (exposure, contrast, white balance).
  • Apply LUTs (Look-Up Tables) to enhance mood.
  • Match colors between clips for consistency.
  • Learn color wheels and curves for grading.
  • Use scopes (waveform, vectorscope) for accurate adjustments.

5. Background Editing and Effects

  • Separate and clean audio (noise reduction, EQ).
  • Sync background music with visuals.
  • Use keyframes to fade in/out audio.
  • Add sound effects (whoosh, ambient sounds).
  • Balance voice, music, and effects (audio mixing).
